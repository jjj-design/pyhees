[[住戸全体の暖房設備オブジェクト]]
== 住戸全体の暖房設備オブジェクト

=== 概要

住戸全体の暖房設備を表すオブジェクトです。次の表のキーから構成されます。

[width="100%",cols="30%, 20%, 50%",options="header",]
|===
|キー |型 |意味
|type |文字列 |暖房設備機器の種類
|duct_insulation |文字列 |ダクトが通過する空間
|VAV |真偽値 |VAV方式
|general_ventilation |真偽値 |全般換気機能
|EquipmentSpec |文字列 |機器の仕様の入力
|q_hs_rtd_H |数値 |定格暖房能力試験の能力
|P_hs_rtd_H |数値 |定格暖房能力試験の消費電力
|V_fan_rtd_H |数値 |定格暖房能力試験の風量
|P_fan_rtd_H |数値 |定格暖房能力試験の室内側送風機の消費電力
|q_hs_mid_H |数値 |中間暖房能力試験の能力
|P_hs_mid_H |数値 |中間暖房能力試験の消費電力
|V_fan_mid_H |数値 |中間暖房能力試験の風量
|P_fan_mid_H |数値 |中間暖房能力試験の室内側送風機の消費電力
|V_hs_dsgn_H |数値 |設計風量
|===

=== キー

==== type キー

暖房設備機器の種類は次の表の値を指定します。型は文字列で指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|ダクト式セントラル空調機 |ダクト式セントラル空調機
|===

==== duct_insulation キー

ダクトが通過する空間は次の表の値を指定します。型は文字列で指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|全て断熱区画内である |全て断熱区画内である

|全てもしくは一部が断熱区画外である
|評価しない、または全てもしくは一部が断熱区画外である
|===

==== VAV キー

VAV方式は次の表の値を指定します。型はbooleanで指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|false |採用しない
|true |採用する
|===

==== general_ventilation キー

全般換気機能は次の表の値を指定します。型はbooleanで指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|false |全般換気機能なし
|true |全般換気機能あり
|===

==== EquipmentSpec キー

機器の仕様の入力は次の表の値を指定します。型は文字列で指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|入力しない |入力しない（規定値を用いる）

|定格能力試験の値を入力する |定格能力試験の値を入力する

|定格能力試験と中間能力試験の値を入力する
|定格能力試験と中間能力試験の値を入力する
|===

==== q_hs_rtd_H キー

定格暖房能力試験の能力は整数で指定します。

==== P_hs_rtd_H キー

定格暖房能力試験の消費電力は整数で指定します。

==== V_fan_rtd_H キー

定格暖房能力試験の風量は小数で指定します。

==== P_fan_rtd_H キー

定格暖房能力試験の消費電力は整数で指定します。

==== q_hs_mid_H キー

中間暖房能力試験の能力は整数で指定します。

==== P_hs_mid_H キー

中間暖房能力試験の消費電力は整数で指定します。

==== V_fan_mid_H キー

中間暖房能力試験の風量は小数で指定します。

==== P_fan_mid_H キー

中間暖房能力試験の室内側送風機の消費電力は整数で指定します。

==== V_hs_dsgn_H キー

設計風量は小数で指定します。

=== 住戸全体の暖房設備オブジェクトの例

----
"H_A":{
  "type":"ダクト式セントラル空調機",
  "duct_insulation":"全てもしくは一部が断熱区画外である",
  "VAV":false,
  "general_ventilation":true,
  "EquipmentSpec":"定格能力試験と中間能力試験の値を入力する",
  "q_hs_rtd_H":7100,
  "P_hs_rtd_H":2240,
  "V_fan_rtd_H":1000.0,
  "P_fan_rtd_H":1500,
  "q_hs_mid_H":3550,
  "P_hs_mid_H":1120,
  "V_fan_mid_H":750.0,
  "P_fan_mid_H":560
}
----


[[主たる居室の暖房設備オブジェクト]]
== 主たる居室の暖房設備オブジェクト

=== 概要

主たる居室の暖房設備を表すオブジェクトです。次の表のキーから構成されます。

[width="100%",cols="30%, 20%, 50%",options="header",]
|===
|キー |型 |意味
|type |文字列 |暖房設備機器または放熱器の種類
|mode |文字列 |運転モード (自立拡張)
|e_class |文字列 |エネルギー消費効率の区分
|dualcompressor |真偽値 |小能力時高効率型コンプレッサー
|e_rtd_H |数値 |定格能力におけるエネルギー消費効率
|r_up |数値 |上面放熱率（床の断熱）
|r_Af |数値 |敷設率
|r_dash_Af |数値 |仮想床の床面積を除いた敷設率
|pipe_insulation |真偽値 |断熱配管の採用
|===

=== キー

==== type キー

暖房設備機器または放熱器の種類は次の表の値を指定します。型は文字列で指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|ルームエアコンディショナー |ルームエアコンディショナー

|FF暖房機 |FF暖房機

|温水暖房用床暖房 |温水暖房用床暖房

|電気ヒーター床暖房 |電気ヒーター床暖房

|ルームエアコンディショナー付温水床暖房機
|ルームエアコンディショナー付温水床暖房機

|温水暖房用パネルラジエーター |温水暖房用パネルラジエーター

|温水暖房用ファンコンベクター |温水暖房用ファンコンベクター

|電気蓄熱暖房器 |電気蓄熱暖房器

|設置しない |暖房設備機器を設置しない

|その他 |その他の暖房設備機器
|===

==== mode キー

運転モードを次の表の値から指定します。型は文字列です。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|連続 |居室連続運転を行います。
|間歇 |居室間歇運転を行います。
|===

==== e_class キー

エネルギー消費効率の区分は次の表の値を指定します。型は文字列で指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|い |区分（い）
|ろ |区分（ろ）
|は |区分（は）
|===

==== dualcompressor キー

小能力時高効率型コンプレッサーは次の表の値を指定します。型はbooleanで指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|false |評価しない、または搭載しない
|true |搭載する
|===

==== e_rtd_H キー

定格能力におけるエネルギー消費効率は小数で指定します。

==== r_up キー

上面放熱率（床の断熱）は小数で指定します。

==== r_Af キー

敷設率は小数で指定します。

==== r_dash_Af キー

仮想床の床面積を除いた敷設率は小数で指定します。

==== pipe_insulation キー

断熱配管の採用は次の表の値を指定します。型はbooleanで指定します。

[width="100%",cols="30%,70%",options="header",]
|===
|値 |意味
|false |評価しない、または採用しない
|true |採用する
|===

=== 主たる居室の暖房設備オブジェクトの例

----
"H_MR": {
  "type": "ルームエアコンディショナー",
  "mode": "間歇",
  "e_class": "は",
  "dualcompressor": false
}
----

|hybrid_category |文字列

|package_id |数値
|電気ヒートポンプ給湯機（CO2冷媒）（太陽熱利用設備を使用しないもの）または電気ヒートポンプ・ガス瞬間式併用型給湯機または電気ヒートポンプ・ガス瞬間式併用型給湯温水暖房機の品番

|pipe_insulation |真偽値 |断熱配管の採用
