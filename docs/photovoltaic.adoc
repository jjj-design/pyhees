[[太陽光発電設備の配列]]
== 太陽光発電設備の配列

=== 概要

太陽光発電設備の集合を表す配列です。
方位の異なる太陽光パネルの面数分の要素を指定します。

=== 要素

==== パネル 要素

方位の異なる太陽光パネルの仕様を表す<<パネルオブジェクト>>です。

    "etr_IN_r":0.927,
    "pv_type":"結晶シリコン系",
    "pv_setup":"屋根置き型",
    "P_alpha":6.283185307179586,
    "P_beta":0.5235987755982988
  },
  {
    "P_p_i":2.0,

  }
]
